# Random Password Generator

Welcome to the Random Password Generator project! This Python script generates strong and random passwords to enhance your online security. Follow the steps below to run the script on your local machine.

## Prerequisites

Before you start, ensure you have the following:

- Python installed on your machine (preferably Python 3).

## Getting Started

1. **Clone the Repository:** Use `git clone` to clone this repository to your local machine.

   ```bash
   git clone https://github.com/parmar-vatsal/Random-Password-Generator.git

2. **Navigate to the Project Directory:**

   ```bash
   cd Random-Password-Generator

3. **Run the Python File:**

   ```bash
   python Random_password_Generator.py
